## Guia de Compilação no Android Studio:

Para que você possa executar e testar o aplicativo em seu dispositivo ou emulador, siga os passos abaixo. O processo é simples, exigindo apenas o Android Studio e o SDK do Android configurado:

1.  **Pré-requisito:** Certifique-se de ter o **Android Studio** instalado (recomenda-se a versão estável mais recente) e que a configuração do SDK do Android esteja completa em sua máquina.
2.  **Obter o Código:** Faça o download do código-fonte ou clone este repositório para uma pasta local.
3.  **Abrir no Android Studio:** Inicie o Android Studio. Na tela inicial, selecione "Open an existing Android Studio project" e navegue até a pasta clonada.
4.  **Sincronização do Gradle:** O Android Studio iniciará a sincronização das dependências automaticamente. Aguarde até que o processo do Gradle seja concluído (você pode acompanhar o progresso na barra de status inferior). Se necessário, force a sincronização clicando no ícone "Sync Project with Gradle Files" (o ícone do elefante).
5.  **Execução:**
    * Na barra de ferramentas superior, selecione um dispositivo de destino (um Emulador Android Virtual Device - AVD - ou um dispositivo físico conectado via USB Debugging).
    * Clique no botão **Run 'app'** (o ícone de 'Play' verde).
    * O ambiente de desenvolvimento fará a construção (build) do APK e, em seguida, instalará e iniciará o aplicativo no dispositivo escolhido.
